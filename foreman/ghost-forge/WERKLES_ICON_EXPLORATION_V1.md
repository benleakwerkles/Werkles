## Three primary families

| Family | ID | Idea | Best for |
|--------|-----|------|----------|
| **Operator Marks** | `operator` | Independent shapes join into one structure; subtle W in negative space | App icon, trust/partnership positioning |
| **Network Marks** | `network` | Nodes + paths form geometric W; discovery graph | Lane UI, connector-heavy surfaces |
| **Foundry Marks** | `foundry` | Anvil + blueprint line reduced to modern mark | Workshop brand spine, header mark alt |

## Five-piece experiment

`five-piece-assembly/` — five lane strokes assemble into readable **W** (Spark, Builder, Operator, Backer, Connector).

- `-copper.svg` — warm cream fill + copper stroke (`#f6efe5` / `#9f6633`)

## Recommendation (v1)

**Strongest direction: Network Marks**

| Criterion | Why Network wins |
|-----------|------------------|
| 24px legibility | Node + stroke grammar survives reduction; no interior detail loss |
| Lane set cohesion | Five lanes map to node roles without pictorial clichés |
| Werkles product fit | Partner discovery + trust network reads immediately |
| Monochrome | Paths stay distinct without fill dependency |
| Favicon | `network-mark-primary` reduces to 3 nodes + 2 strokes cleanly |

**Runner-up:** Operator Marks — premium software feel, best for **app icon** if brand leans partnership-over-graph.  
**Foundry Marks** — strongest workshop spine for **header alt**, weakest at 16px (anvil base compresses).

## Event

Ben manually carried packets, art direction, copy drafts, and red-team rulings between GD, cousins, and Maker.

## Context

GD can classify and generate packets but courier maturity lags routing maturity.

## Decision

Build GD Command Console / GimpDash to reduce packet-mule work — consult Speaker, stop before Send.

## Why it happened

No single cockpit surface; each Aeye held a slice of truth outside the repo.

## Risk exposed

Routing improves while delivery still fails — Ben remains switchboard.

## Lesson learned

Ben should not be the message bus. Automatica means GD routes + registry locates + courier prepares + Speaker supplies memory + Ben gates meaningful decisions only.

## Doctrine changed

`GD_COMMAND_CONSOLE_V0` — human-consumable output rule applies to all dispatch synthesis.

## Who must remember

GD, Foreman control server, Operator, Relay Courier builders.

## Future warning

Interrupt when a mission "completes" with repo files but no Operator Brief or clipboard path.

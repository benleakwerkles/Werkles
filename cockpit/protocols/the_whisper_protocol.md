# The Whisper Protocol & Eyewatch Array

## 1. The Subconscious Ledger
* **Directory:** `/repo/nervous_system/whispers/`
* **The Rule:** Intuiters log unverified `.md` hunches here. Whispers carry a 48-hour half-life (Negative Feedback Decay). If a Listener does not validate a whisper within 48 hours, Ender automatically excises it to keep the context sterile. A whisper cannot trigger a Responder.

## 2. The Eyewatch Sensory Arrays
* **Wormeyes (Micro):** Watches local Betsy physical state—file saves, syntax errors, and port collisions (`localhost:3000`). Drops `WORM_WHISPER.md`.
* **Birdeyes (Macro):** Watches fleet coordination—PR merge velocity, task queue bottlenecks, and handoff loops. Drops `BIRD_WHISPER.md`.
* **Flyeyes (External):** Watches the boundary—external API payloads, Squibb scrapes, and incoming user intake artifacts. Drops `FLY_WHISPER.md`.
* **Swatters (Reflex):** Intercepts confirmed Worm/Fly whispers that match the Top 25 Pre-Approval Policies. Executes the fix instantly; logs to `/repo/nervous_system/receipts/`.

## 3. Quorum Sensing (Action Potential)
* Responders are kept dormant until the aggregated Whisper "Frictional Heat Score" hits 100.

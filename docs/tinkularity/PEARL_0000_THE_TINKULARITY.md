## Core Doctrine

The organism must load its own memory before it acts.

Ben is the Operator, not the transport layer, not the clipboard, not the context mule, and not the manual bootloader. If an Aeye needs the Book, the frontier, the current repo reality, or the shared state, the machine must fetch it from the repo.

TinkerDen is a command center, not a chat surface. Mission Control explains why. Bridge shows now. Engine Room shows execution. Speaker preserves causal memory. Wormeyes reports physical reality. The Corpus Callosum keeps Aeyes aware of shared frontier state.

Escape velocity means the system can end a session and wake back up with its doctrine, frontier, shared state, and local reality physically reloaded from disk.
